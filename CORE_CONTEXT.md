## Purpose

Generic enrichment framework that bridges managed records (SDS, GBU, tenders, trades) with external knowledge sources (GESTIS, PubChem, ECHA, market APIs). Pure Python core, optional Django integration.

## Tech Stack

- **Python** 3.12+
- **httpx** — HTTP client (replaces requests)
- **tenacity** — retry with exponential backoff
- **hishel** — HTTP response caching (SQLite-backed)
- **pyrate-limiter** — per-provider rate limiting
- **Django 5.x** — optional integration via `EnrichableModelMixin`
- **Build**: hatchling
- **Tests**: pytest + respx (HTTP mocking)
- **Lint**: ruff

## Architecture

```
EnrichmentProvider (Protocol)
    ↓
HTTPProviderBase (_base.py)     ← thread-safe lazy httpx.Client
    ↓                               ↓
GESTISProvider                  PubChemProvider
    ↓                               ↓
EnrichmentRegistry              ← central dispatch, domain-scoped
    ↓
EnrichableModelMixin (Django)   ← JSONB storage in enrichment_data
```
